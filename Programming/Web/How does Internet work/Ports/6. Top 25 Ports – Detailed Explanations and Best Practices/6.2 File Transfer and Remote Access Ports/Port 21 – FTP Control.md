**Usage:** Command channel for FTP sessions.

**Threats:**
- Credential sniffing and brute-force attacks
- Unauthorized access if left open

**Best Practices:**
- Transition to FTPS or SFTP
- Disable anonymous logins
- Use IP filtering and rate limiting
