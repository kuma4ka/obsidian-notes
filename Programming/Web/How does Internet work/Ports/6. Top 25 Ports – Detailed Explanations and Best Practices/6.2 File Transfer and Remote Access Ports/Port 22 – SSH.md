**Usage:** Secure remote shell, SFTP, and tunneling.

**Threats:**
- Brute-force attacks
- Weak cipher suites
- Misconfigurations

**Best Practices:**
- Enforce key-based authentication; disable passwords
- Change default port if possible
- Use rate-limiting tools (Fail2Ban)
- Keep SSH software updated
